# Heart Disease Prediction System 💔 Scale Model

## 📌 Project Overview
Developed a machine learning classification model to predict the presence of heart disease based on various clinical health attributes (such as age, sex, chest pain type, blood pressure, cholesterol, and fasting blood sugar). This tool showcases how predictive analytics can assist medical professionals in early-stage cardiovascular diagnoses.

## 🛠️ Tech Stack & Libraries
- *Language:* Python
- *Environment:* Jupyter Notebook
- *Libraries Used:* - *Data Manipulation:* Pandas, NumPy
  - *Data Visualization:* Matplotlib, Seaborn
  - *Machine Learning:* Scikit-Learn

## 🚀 Key Features & Pipeline
- *Exploratory Data Analysis (EDA):* Visualized distribution trends, attribute correlations, and target balance across the health metrics using Seaborn heatmaps and pairplots.
- *Data Preprocessing:* Handled missing data, scaled numerical features, and encoded categorical health indicators for baseline compatibility.
- *Model Selection & Training:* Evaluated and cross-validated classification models (such as Logistic Regression / Random Forest) using Scikit-Learn to optimize accuracy.
- *Evaluation Metrics:* Analyzed performance using Confusion Matrices, Classification Reports (Precision, Recall, F1-Score), and ROC-AUC curves.

## 📊 How to Run the Project
1. Clone this repository.
2. Ensure you have the dependencies installed: pip install pandas numpy scikit-learn matplotlib seaborn
3. Open `Heart Decision Prediction ` in Jupyter Notebook and execute the cells.
